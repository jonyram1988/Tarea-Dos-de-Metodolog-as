EXPLICACIÓN

La presente tarea esta consumiendo un JSON público del Ministerio de Salud de Chile y que es muy conocido y utilizado llamado Farmacia de Turno, este servicio se obtiene los listados de farmacias a través del territorio nacional obtenido de Farmanet (MINSAL). La idea es demostrar la utilización JSON y la manipulación del DOM mostrando los resultados en una simple tabla HTML.

Estoy ocupando Bootstrap 3 para hacer más fácil el Front-End y utilizando la libreria jQuery para las llamadas Ajax.

Finalmente mencionar que por mótivos prácticos la búsqueda solamente se restringe a la Octava Región y para el día 01/05/2018.
